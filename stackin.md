### 2.3 Why the asymmetric verse lengths

V1 = 16 bars (full documentary arc requires the full catalog before the lens can break). V2 = 8 bars (phonk-canonical brevity; V2 executes its job by L8 — confession lands, dime callback closes, tape stop carries the exit). Half the bars, full arc, mirrored form. Extending V2 to 16 would dilute the hit. The asymmetry IS the structure.

## 3. Beat Spec — Memphis Phonk

### 3.1 Tempo / key / mode
- **140 BPM**, half-time perceived ~70
- **D minor** (alt: A minor)
- Natural minor + tritone (Ab) on lead lines for menace

### 3.2 Drum kit
| Element | Spec |
|---|---|
| Kick | Lo-fi 808 kick, tape-saturated, ~80 Hz body, slight pitch wobble (DJ Screw / Three 6 era) |
| Snare | Saturated trap snare layered with vinyl crackle. 200 Hz body, top-end rolled off above 6 kHz (no modern crack) |
| Clap | OFF or very low — Memphis is snare-driven |
| Closed hat | Lo-fi 1/8 pulse, tape-saturated, narrow stereo, LOW density vs Atlanta |
| **Cowbell** | **Required.** Off-beats. 800 Hz–1.5 kHz body. Three 6 / Tommy Wright signature |
| Vinyl crackle | Constant low-level layer at -30 dB. Source: free vinyl crackle audio loop on dedicated audio track. |

### 3.3 808 design
| Param | Value |
|---|---|
| Tuning | D1 root |
| Glide | 50–100ms (minimal — Memphis is staccato/drone, not slide) |
| Saturation | Stock chain: Saturator (Soft Sine, Drive 4–6 dB) → Redux (Bit 14, Downsample 2) → EQ Eight. Full settings in §4.5.5. |
| Pattern | Sparse. Root on 1, root on 3 most bars. Walk every 4–8 bars. NOT melodic 808 lines. |
| Layering | Sub layer (sine, 30–50 Hz) + saturated mid. Skip distortion grit layer. |

### 3.4 Sample / melodic loop
**Source:** Pitched-down minor organ + female vocal chop, vinyl-saturated. 4-bar loop, repeats throughout.
**Why this pick:** "Intercom dims behind him, soft and so kind" — the loop literally IS the intercom fading. Self-mirroring imagery.
**Alternates:** dark string section with tritone hits, or lo-fi piano loop.

### 3.5 Arrangement (~2:00, full song flow)
```
Intro (8):    Sample chop + tape hiss + occasional cowbell
Drop 1 (16):  Full kit + 808 + sample. VERSE 1.
Break (4):    Drums strip, sample + 808 only
Hook (8):     Full kit returns, slight hat density bump
Drop 2 (8):   VERSE 2 (half-length)
Hook (8):     Final, vocal chop ad-libs over
Outro (4):    Tape stop + filter sweep, sample bleeds out
```

Total ~64 bars at 140 BPM half-time = ~1:50–2:00.

### 3.6 Mix targets
| Element | Target |
|---|---|
| Integrated LUFS | -9 to -7 |
| True peak | -1.0 dBTP |
| Kick | 80 Hz body, lo-fi rolloff above 8 kHz |
| 808 sub | 50 Hz fundamental, saturated mids 200–600 Hz |
| Snare | 200 Hz body, top rolled off above 6 kHz |
| Cowbell | 800 Hz–1.5 kHz, panned slightly off-center |
| Hat | 8 kHz top, narrow stereo |
| Master bus | Stock chain: Saturator (Soft, Drive 2–3 dB) → Glue Compressor (Ratio 2:1, Attack 10ms, Release Auto, GR 1–2 dB) → Limiter (-1.0 dBTP). Full settings in §4.5.6. |

**Vocal-specific:**
- Vocal forward (-3 to -1 dB below loudest sample peak) — documentary verse needs intelligibility
- Light tape sat on vocal bus
- V1 L1–L8 dense — *don't* clutter with delay/reverb during these bars
- Hook vocal can have 1/8-dotted delay or short plate to lift the chant

### 4.5 Ableton 12 implementation

#### 4.5.1 Project architecture

**Track layout:**

| Track # | Type | Contents | Output routing |
|---|---|---|---|
| 1 | MIDI → Drum Rack | Kit: kick (pad C1), snare (D1), closed hat (F#1), cowbell (A#1) | → Drum Bus |
| 2 | MIDI → Operator (or Simpler) | 808 — mono mode for any pitch glide | → 808 Bus |
| 3 | Audio | Vinyl crackle loop — 8-bar audio clip looped at -30 dB | → Drum Bus |
| 4 | MIDI → Sampler | Melodic loop / sample chop — pitched-down minor organ + female vocal chop | → Master |
| 5 | Audio (or Return) | Hook vocal ad-libs — pitched-down "tatters... tatters..." chops | → Master |
| 6 | Audio | Lead vocal | → Vocal Bus |

**Bus structure:**

```
Drum Bus  ─┐
           ├──→ Master Bus ──→ Limiter → Out
808 Bus   ─┤
Vocal Bus ─┤
Sample    ─┘
```

- **Drum Bus** = Drum Rack + crackle. Light glue compression (1–2 dB GR), tape sat
- **808 Bus** = isolated for parallel saturation chain
- **Vocal Bus** = light tape sat + delay/plate sends for hook only
- **Master Bus** = final glue + saturation + limiter

**Why 808 lives outside the Drum Rack:** mono operation (for glide), independent saturation chain, ability to write melodic 808 patterns later.

#### 4.5.2 Swing setup (Groove Pool walkthrough)

1. **Open Groove Pool:** wave icon at bottom of main window, or View menu → Show Groove Pool
2. **Find groove:** Browser → Categories → Grooves → MPC60 folder → `MPC60-1-16.agr`
3. **Load:** drag into Groove Pool panel
4. **Apply to drum clip:** select MIDI clip → Clip View → Groove dropdown → set to `MPC60-1-16`
5. **Tune sliders:**

| Slider | Value | Why |
|---|---|---|
| Base | 1/16 | Matches groove's grid resolution |
| Quantize | 0% (off) | Don't re-quantize after applying groove |
| **Timing** | **60–65%** | ≈ 58% off-beat shuffle (Memphis canonical) |
| **Random** | **3–5%** | Micro-humanization. Higher = sloppy. |
| Velocity | 0% | You'll handle velocity manually (see 4.5.8) |

6. **Optional commit:** right-click clip → Commit Groove. Don't commit until final bounce.

**Tuning:**
- Too floppy / late → drop Timing to 55%
- Too straight / robotic → push Timing to 70%

#### 4.5.3 1/16T roll — Method A (manual MIDI, deterministic)

For studio production / consistent loops:

1. Open the drum clip's MIDI editor
2. Zoom to last 16th of bar 4
3. Switch grid resolution to 1/48 (right-click ruler → Fixed Grid → 1/48)
4. Place 3 closed-hat notes within the last 16th, evenly spaced on the 1/48 grid
5. Set velocities: 60, 80, 100 (left-to-right ramp)
6. Verify the 3 hits land *inside* the last 16th

**Result:** every play identical. Predictable.

#### 4.5.4 1/16T roll — Method B (live with controller)

For performance feel / one-pass tracking:

1. Drum Rack hat pad — confirm choke group set to itself
2. Set controller to velocity-sensitive mode (Push 3 default; MPK Mini Pro: confirm pads not in Fixed Velocity)
3. Arm drum track, press record, hit hat pad 3 times rapidly on last 16th
4. Quantize to 1/48 grid, preserve velocity
5. Audition; re-record if velocity ramp didn't land

**When to pick A vs B:** A for production loops. B for live performance or tracking sessions.

#### 4.5.5 808 saturation chain (stock — 808 Bus)

In order on the 808 Bus track:

| Position | Plugin | Settings | Function |
|---|---|---|---|
| 1 | **Saturator** | Type: Soft Sine. Drive: 4–6 dB. Output: -2 dB. DC switch: ON. | Harmonic content, gentle compression-like behavior |
| 2 | **Redux** | Bit: 14. Downsample: 2. | Lo-fi character without obvious artifacting |
| 3 | **EQ Eight** | Low shelf +2 dB at 80 Hz. Bell cut -2 dB at 250 Hz (Q 0.7). High shelf cut -3 dB at 6 kHz. | Sub bump, mud removal, top roll-off |

**Tuning:**
- Saturator Drive: 4 dB subtle, 6 dB audible-musical, 8+ dB too aggressive for Memphis
- If 808 loses fundamental, reduce Drive and increase EQ low-shelf
- Try Analog Clip mode on Saturator for harder character

#### 4.5.6 Master bus chain (stock — Master)

| Position | Plugin | Settings | Function |
|---|---|---|---|
| 1 | **Saturator** | Type: Soft. Drive: 2–3 dB. Output: -1 dB. | Subtle harmonic glue |
| 2 | **Glue Compressor** | Threshold to 1–2 dB GR on loudest sections. Ratio: 2:1. Attack: 10ms. Release: Auto. | Mix cohesion |
| 3 | **Limiter** | Ceiling: -1.0 dBTP. Gain: push to -8 to -7 LUFS integrated. | Final loudness ceiling |

**Order matters:** saturation BEFORE compression. Saturation generates harmonics; compression shapes the result.

**LUFS targeting:** integrated -8 to -7 LUFS (phonk standard). Don't mix to streaming targets (-14 LUFS Spotify) — those are playback normalization, not master targets.

#### 4.5.7 Snare/hat lo-fi processing (Drum Rack pad inserts)

| Element | Plugin | Settings |
|---|---|---|
| Snare pad insert | **Redux** | Bit: 12. Downsample: 4–6. Effective sample rate ~16k. |
| Closed hat pad insert | **Redux** | Bit: 12. Downsample: 4. Slightly less aggressive than snare. |
| Cowbell pad insert | (none) | Cowbell stays cleaner. Light Saturator if needed. |

#### 4.5.8 Velocity workflow — Method A (MIDI editor, deterministic)

1. Open drum clip MIDI editor
2. Use velocity lane (bottom strip)
3. **Hat humanization:** select all hat notes → MIDI tools → Velocity → Randomize → ±15%
4. **Roll velocity ramp:** set 3 roll hits to 60, 80, 100 manually
5. **Snare accents:** main snare on 3 = 100. Last-16th pre-flam = 70.
6. **Kick:** uniform 100, slight randomization ±5%
7. **Cowbell:** ±10% randomization

#### 4.5.9 Velocity workflow — Method B (live controller)

1. Use velocity-sensitive controller (Push 3, MPK, Maschine)
2. Confirm Fixed Velocity OFF
3. Record drum performance live
4. Quantize timing only, preserve velocity (Edit → Quantize Settings → Velocity Amount = 0%)
5. Clean up missed hits in MIDI editor while keeping captured velocities
6. Optionally apply MPE if controller supports (Push 3) — pad pressure → filter/pitch in Drum Rack

**Hybrid recommendation:** live record (B) → MIDI editor cleanup (A). Best of both.

### 4.6 Pocket considerations (drum × vocal)
- V1 L7 is the densest bar (12 syl). **Drop cowbell hits during V1 L7 only** — let the bar breathe.
- V1 L9 is short (7 syl) and held. **Strip hat hits during V1 L9** — beat carries kick + 808 + cowbell. Vocal sits in gap.
- Hook section: full hat density returns. Hook is hypnotic — drums chant alongside the voice.
- V2 L6 is the [push] bar (urgency) — keep hat density steady but consider dropping cowbell on this single bar to clear runway.
- V2 L8 outro builds — tape stop begins under "dime" sustain, drums fade with sample.

## 7. Production Checklist (Release-Ready)

Phased build with completion gates. ADHD-context: don't skip phases, don't move to next phase until current gate is met.

### 7.1 Vocal capture chain

**Owned gear:** Zoom H6 with MSH-6 capsule. Prior diagnosis: excessive preamp gain + untreated room. Both fixable without new purchase.

| Step | Setting / Action |
|---|---|
| Mic capsule | MSH-6 acceptable. **Better:** Shure SM7B (~$400) or Rode NT1 (~$250) + Focusrite Scarlett 2i2 (~$180). Wishlist note. |
| H6 input | XY capsule pointed at mouth, ~6–8" distance. MSH-6 alternate: position mid-capsule cardioid toward mouth. |
| Preamp gain | **CRITICAL:** target peak -12 to -6 dBFS on H6 meter. Stay below ~70% gain (preamp noise increases above). |
| Room treatment | Vocal booth: closet with hanging clothes, OR mattress propped behind mic, OR moving blankets on stands forming 3-sided enclosure. Sit ~3 feet from any hard surface. |
| Pop filter | Required. DIY: pantyhose stretched over wire hanger 4" from mic. |
| Monitoring | 64 Audio A12t IEMs (owned) — closed isolation. Send vocal pre-FX through H6 monitoring at low level. |
| File format | 24-bit / 48 kHz WAV. |
| Take protocol | 3 full takes minimum, doubles for hook (4 layers). Comp later in Ableton. |

**Gate:** vocal takes have NO audible room reverb tail, NO hiss above -50 dBFS, NO clipping. Re-track if any present.

### 7.2 Vocal processing chain (Ableton stock + Auto-Tune Unlimited)

In order on Vocal Bus:

| Position | Plugin | Settings | Function |
|---|---|---|---|
| 1 | **Gate** | Threshold: -45 dB. Attack: 1ms. Hold: 10ms. Release: 100ms. | Cut room noise between phrases |
| 2 | **EQ Eight** (HP) | High-pass at 80 Hz, 24 dB/oct | Remove sub rumble |
| 3 | **Auto-Tune Unlimited** | Documentary verses (V1 L1–9, V2 L1–4): Retune speed 30–50 (transparent). Key: D minor. <br> Indictment + testimony (V1 L10–16, V2 L5–8): Same transparent setting. <br> Hook: Retune speed 20–30 (slightly tighter — chant feel, not T-Pain). | Phonk-appropriate transparent pitch correction |
| 4 | **Compressor** | Threshold for 4–6 dB GR on loudest phrases. Ratio 3:1. Attack 5ms. Release Auto. | Tame dynamics for documentary intelligibility |
| 5 | **EQ Eight** (corrective) | Cut 200–400 Hz mud (-2 dB, Q 0.7). Cut 2.5–4 kHz harshness if needed. Boost 8–10 kHz +1 dB for air. | Vocal seat in mix |
| 6 | **De-Esser** | Multiband Dynamics in De-Ess preset. Frequency 5–8 kHz. | Tame "s" / "sh" peaks |
| 7 | **Saturator** | Type: Soft. Drive: 1–2 dB. | Match beat aesthetic |

**Sends (parallel):**
- Send 1: **Plate reverb** (Ableton Reverb, Plate, 1.2s decay, pre-delay 30ms) — only on hook bars. Send: -18 dB. Bypass during V1 L1–L8.
- Send 2: **1/8-dotted delay** — only on hook bars and last word of each verse-section. Send: -22 dB. Feedback 25%. Filter low-cut at 300 Hz.

**Hook layering (4 stacked vocals):**
1. Main vocal (lead, center)
2. Double (panned ~10 L)
3. Octave-down (panned ~10 R, low-pass at 4 kHz, -6 dB)
4. Ad-libs (`tatters... tatters...` whispered, panned wide L/R alternating)

### 7.3 Reference tracks (A/B during mix)

Listen at matched volume. Match low-end weight, vocal forwardness, top-end roll-off character.

| Tier | Reference | Use case |
|---|---|---|
| Memphis canonical | Three 6 Mafia (early), Tommy Wright III, DJ Paul + Juicy J early | Tonal target — lo-fi tape, narrow stereo, dark mid-range |

**A/B protocol:** every mix decision, 30 sec your track / 30 sec reference / 30 sec your track, matched volume.

### 7.5 Mastering protocol

**Self-master OR send out?** Self-master is fine for first single; pro-master if budget allows ($30–80 typical for phonk).

**Self-master chain (Ableton stock):**

| Position | Plugin | Settings |
|---|---|---|
| 1 | **EQ Eight** (corrective) | HP at 25 Hz. High-shelf cut -1 dB at 12 kHz if too bright. |
| 2 | **Saturator** | Type: Soft. Drive: 2–3 dB. |
| 3 | **Glue Compressor** | Ratio 2:1. Attack 10ms. Release Auto. GR 1–2 dB. |
| 4 | **EQ Eight** (final tonal) | Optional shelf: +0.5 dB at 8 kHz, +0.3 dB at 60 Hz. |
| 5 | **Limiter** | Ceiling: -1.0 dBTP. Gain push to -8 to -7 LUFS integrated. |

**Verification before bounce:**
- Integrated LUFS: -8 to -7
- Short-term LUFS max: -6
- True peak: -1.0 dBTP (no overs)
- Sample peak below -1.0 dBFS after limiter

**Distribution format:**
- Spotify / Apple / YouTube (via DistroKid, TuneCore): 16-bit / 44.1 kHz WAV, dithered from 24-bit
- Bandcamp: 24-bit / 48 kHz WAV
- SoundCloud: 16-bit / 44.1 kHz WAV or 320 kbps MP3
- Backup: keep 24-bit / 48 kHz master indefinitely

**Dither:** when bouncing 16-bit, enable POW-r 2 dither.

### 7.6 Session order (phased build with gates)

**Phase 1 — Beat skeleton** (1–2 sessions)
- Build Drum Rack (kick + snare + hat + cowbell)
- Lay 4-bar drum pattern (§4.1) + 1 bar fill (§4.2)
- 808 on separate track with saturation chain (§4.5.5)
- Source + chop sample loop (§3.4)
- Apply Groove Pool (§4.5.2)
- **Gate:** loop sounds like phonk for 8 bars without getting boring.

**Phase 2 — Beat arrangement** (1 session)
- Build full song structure (§3.5)
- Add tape stop, filter sweeps, perc additions (§4.4)
- **Gate:** instrumental sustains 2 minutes solo.

**Phase 3 — Vocal recording** (1–2 sessions)
- Set up capture chain (§7.1)
- Track lead vocal: 3 full takes
- Track hook layers (4 stacks)
- Track ad-libs (V2 "declined" echo, "dime" pitched-down decays)
- **Gate:** clean takes (no clipping, no room reverb, no excessive hiss).

**Phase 4 — Vocal editing** (1 session)
- Comp best takes per phrase
- Time-align doubles
- Tune through Auto-Tune Unlimited (§7.2 step 3)
- Apply full vocal processing chain (§7.2)
- **Gate:** vocals sit in pocket, intelligible, no obvious tuning artifacts.

**Phase 5 — Mixing** (2 sessions)
- Balance levels (start with vocal)
- Apply drum bus + 808 bus + master bus chains
- A/B against references (§7.3)
- Sends (plate, delay) for hook only
- **Gate:** mix translates on at least 3 systems (IEMs, phone speaker, car).

**Phase 6 — Mastering** (1 session)
- Apply mastering chain (§7.5)
- Verify LUFS, true peak, sample peak
- Bounce 24-bit master + 16-bit/44.1 distribution copy
- **Gate:** master matches reference loudness without obvious distortion.

**Phase 7 — Stems + metadata + release** (1 session)
- Export stems (§7.4)
- Fill release metadata (§7.7)
- Upload to distributor
- **Gate:** all files archived, distributor confirms ingestion.

**Anti-hyperfocus reminder:** if any phase exceeds 2 sessions, stop. 85% rule — get to 85%, move on. Polish in mastering, not earlier phases.
